2\. ¿Qué es un fork en GitHub y cuándo se usa?

fork es una copia independiente de un repositorio que se aloja en tu propia cuenta de GitHub

permitiéndote experimentar con el código libremente y poder ofrecer ese cambio al proyecto original sin afectar el repositorio



diferencia de fork y clon



la diferencia principal que posee el fork es que este te permite tener una copia en tu cuenta de GitHub a diferencia de clon que descarga la copia directamente en tu ordenador



en que contribuiría en un proyecto open source

permite poder experimentar con el código si nesedidad de alterar el proyecto original
