¿Qué es Git Flow? 



Git Flow es una forma de organizar las ramas de un proyecto para que todo esté ordenado.

Sirve para trabajar en equipo y no dañar la versión estable del proyecto.



¿Qué es un HEAD detached?

Un “HEAD detached” pasa cuando Git se queda en un commit viejo o en uno que no pertenece a ninguna rama.



¿Qué hace git log?

git log sirve para ver el historial de todos los commits que se han hecho en el proyecto
